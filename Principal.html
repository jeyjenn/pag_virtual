Proyecto

Funenaria "San Jose"
Diseño y desarrollo de una aplicación web para la gestión de ventas de los servicios exequiales de la funeraria “San José”, permite llevar el control en sus procesos, gestionando de forma ágil, rápida y eficiente sus ventas, a través de un entorno web fácil de utilizar y accesible desde cualquier lugar que disponga de conexión a internet.

Codificado en lenguaje Java en entorno web, mediante el IDE de NetBeans, utilizando el patron de diseño MVC, gestionado por la base de datos mysql y librerias de Sweetalert, Bootstrap, Datatables.
